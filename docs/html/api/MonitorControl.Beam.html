Beam¶

class MonitorControl.Beam(name)¶

Bases: MonitorControl.Signal

Signal class for radiation arriving at a feed.

There is no more fundamental signal type. A Beam signal has two polarizations in it.

A polarizer extracts one of the polarizations. An orthomode extracts both. The resulting signal(s) are complex, having implicit phase information.
