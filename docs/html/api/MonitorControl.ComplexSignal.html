ComplexSignal¶

class MonitorControl.ComplexSignal(parent_signal, pol=None, name=None)¶

Bases: MonitorControl.Signal

ComplexSignal class for an RF output from an orthomode or polarizer.

This is a complex signal with both in-phase and quadrature-phase components. A simple mixer extracts only the in-phase component. A complex mixer extracts both.
