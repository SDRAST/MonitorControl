DSN_rx¶

class MonitorControl.Receivers.DSN.DSN_rx(name, inputs=None, output_names=None)¶

Bases: MonitorControl.Receivers.Receiver

Class for DSN receivers with no M&C functions

All DSN receivers have a one-to-one input to output mapping. They may have one or two polarization channels, ‘R’ being used for telecom.
